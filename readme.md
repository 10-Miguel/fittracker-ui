Archivos:
index.html: Página principal del proyecto. Contiene el header con navegación, un banner de bienvenida (hero), sección de entrenamientos disponibles, estadísticas resumidas y footer.

profile.html: Página de perfil del usuario. Muestra datos personales como nombre, edad, peso y altura, así como los objetivos de entrenamiento.

trainings.html: Página de entrenamientos. Muestra los distintos tipos de ejercicios disponibles, con duración e intensidad.

stats.html: Página de estadísticas. Resume la actividad del usuario, incluyendo calorías quemadas, entrenamientos completados y tiempo total.

Ejecución:
Para visualizar el proyecto, abrir index.html en cualquier navegador moderno. Todas las páginas están interconectadas mediante enlaces de navegación dentro del header. El archivo main.css contiene todos los estilos y se encuentra dentro de la carpeta styles.
